<h1>osTicket - Prerequisites and Installation</h1>
Hello and welcome! I'm Amy, an IT professional and I'm thrilled to present my first GitHub project and tutorial on setting up osTicket, a powerful open-source help desk ticketing system. If you’re looking to streamline your support communications, you’ve come to the right place!

<h2>What is osTicket?</h2>
osTicket is a highly-reliable open-source ticketing system. It streamlines the management, organization, and archiving of support requests and responses, fostering excellent communication and accountability with your customers.

<h2>Tools and Technologies We Will Employ:</h2>

- Microsoft Azure Virtual Machine (a virtualized computing resource over the cloud)
- Remote Desktop (enabling remote access to the virtual machine)
- Internet Information Services (IIS) - A robust web server solution from Microsoft
- Heidi SQL (an intuitive database management system for streamlined data handling)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)
